API link : https://sportsdata.io/developers/api-documentation/nfl#/endpoint/player-details-by-team 

Details: This application is a Baltimore Ravens player guessing game, there is a new player every 12 hours and you have 8 guesses, each guess will provide details about how close you are to the daily player (such as height and position). The api has a limit on # calls so it will write the data to a file every 24 hours to ensure the data is up to date. User guesses are stored in mongoDB until end of game as well as other users scores for the current time period.
